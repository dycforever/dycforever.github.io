先来个头像

上文中的头像两个字，我用<q>做了引用，而&quot也可以表示引号的:""，虽然形状上还是有点差别。
那么，<q>的意义何在呢？就是从语义上告诉浏览器，这个文字我是在引用的！既然浏览器知道了你的用意，在必要的时候，它自然就会帮你对引用的文字做合适的渲染和处理

对于大段文字的引用，html还提供了另一个工具：<blockquote>，下面看看它的效果：

int find_target(const vector &num, int left, int right, int target) {
    int mid = (left + right) / 2;
    while (left <= right) {
        mid = (left + right) / 2;
        if (num[mid] == target) {
            return mid;
        } else if (num[mid] < target) {
            left = mid+1;
        } else {
            right = mid-1;
        }
    }
    return -1;
}
